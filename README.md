# 🎓 ระบบสารสนเทศ สาขาวิชาภาษาอังกฤษ คณะครุศาสตร์ มหาวิทยาลัยราชภัฏเลย (English Education, LRU)

ยินดีต้อนรับสู่ระบบสารสนเทศและบริการออนไลน์ของสาขาวิชาภาษาอังกฤษ คณะครุศาสตร์ มหาวิทยาลัยราชภัฏเลย พัฒนาขึ้นโดย **Sorasuk Thongkum (@sorasukt)** เพื่ออำนวยความสะดวกแก่นักศึกษาและบุคลากรในสาขาวิชาในการดำเนินกิจกรรมต่าง ๆ ได้อย่างสะดวก รวดเร็ว และโปร่งใส

🌐 **เว็บไซต์:** [https://solution.litalkeducation.com/engedlru/](https://solution.litalkeducation.com/engedlru/)

---

## 🚀 ฟีเจอร์และระบบย่อยภายในโครงการ (Key Features)

โครงการนี้ประกอบด้วยระบบย่อย (Web Applications) 5 ระบบหลัก ดังนี้:

### 1. 🏠 หน้าหลักและศูนย์ข่าวสาร (Dashboard - `index.html`)
*   หน้าแรกที่รวบรวมลิงก์เข้าใช้งานบริการออนไลน์ทั้งหมด
*   สไลเดอร์ประชาสัมพันธ์ข่าวสาร กิจกรรม และคำแนะนำการใช้งานระบบต่าง ๆ
*   แสดงข้อมูลปรัชญาสาขาวิชา และความสำคัญของวิชาชีพครูภาษาอังกฤษ
*   มีระบบแจ้งเตือนข่าวสาร/กิจกรรมสำคัญในรูปแบบ Modal และการแจ้งเตือน PWA

### 2. 📍 ระบบบันทึกการเข้าร่วมกิจกรรมด้วย GPS (`checkup.html`)
*   ระบบสำหรับให้นักศึกษาลงชื่อเช็คชื่อเข้าร่วมกิจกรรมต่าง ๆ ของสาขาวิชา
*   **การยืนยันตัวตน:** เข้าใช้งานด้วยบัญชีอีเมลมหาวิทยาลัย (@lru.ac.th) ผ่าน **Auth0 SPA SDK**
*   **ระบบพิกัด GPS:** ตรวจสอบและเปรียบเทียบพิกัดปัจจุบันของนักศึกษาว่าอยู่ในรัศมีไม่เกิน 100 เมตรจากอาคารเรียนกิจกรรมเพื่อป้องกันการลงชื่อจากนอกสถานที่
*   **คู่มือระบบตำแหน่ง:** มี Accordion แนะนำขั้นตอนการเปิดสิทธิ์ขอเข้าถึง Location สำหรับทั้ง iOS (Safari) และ Android (Chrome)

### 3. 🔑 ระบบค้นหารหัสกิจกรรมและ Token Key (`sys.html`)
*   บริการสืบค้นข้อมูลรหัสกิจกรรมและ Token Key สำหรับนำไปกรอกลงในระบบกิจกรรมของมหาวิทยาลัยราชภัฏเลย
*   ระบบเลือกกิจกรรมและค้นหาด้วยรหัสนักศึกษา 10 หลัก
*   ยืนยันตัวตนก่อนการค้นหาผ่าน Auth0 เพื่อความปลอดภัยของข้อมูล
*   มีปุ่มคัดลอกรหัส (Copy to Clipboard) และปุ่มแชร์ลิงก์อย่างรวดเร็ว

### 4. 💼 ระบบบริหารงบประมาณโครงการ (`budget.html`)
*   ระบบสำหรับบันทึกรายงานการใช้จ่ายเงินโครงการต่าง ๆ ของสาขาวิชา
*   **จำกัดสิทธิ์การเข้าใช้งาน (Role-based Access):** เข้าใช้งานได้เฉพาะผู้รับผิดชอบโครงการที่ได้รับการแต่งตั้งและได้รับสิทธิ์เป็นกรณีพิเศษเท่านั้น (ระบุอีเมลผู้ดูแล)
*   **การบันทึกข้อมูล:** ระบุวันที่ หมวดเงิน (เช่น ค่าจ้างชั่วคราว, ค่าตอบแทน, ค่าใช้สอย, ค่าวัสดุ, ค่าครุภัณฑ์ ฯลฯ) รายละเอียด จำนวน และราคาต่อหน่วย พร้อมระบบคำนวณยอดเงินรวมอัตโนมัติ
*   **หลักฐานการจ่าย:** รองรับการแนบไฟล์หลักฐานใบเสร็จรับเงินหรือเอกสารประกอบการเบิกจ่ายทั้งรูปแบบไฟล์ภาพและ PDF ขนาดสูงสุดไม่เกิน 10MB

### 5. 🗳️ ระบบลงคะแนนเลือกตั้งอิเล็กทรอนิกส์ (`vote/`)
*   ระบบลงคะแนนเลือกตั้งผู้แทนสาขาวิชาภาษาอังกฤษ คณะครุศาสตร์ มหาวิทยาลัยราชภัฏเลย
*   ประกอบด้วยหน้าย่อยที่ทำงานร่วมกัน:
    *   `index.html`: หน้าแรก แสดงผลเวลานับถอยหลังเปิดคูหาและปิดรับคะแนนเสียง
    *   `info.html`: หน้าระบบสำหรับให้นักศึกษาตรวจสอบสิทธิ์ผู้มีสิทธิ์เลือกตั้ง
    *   `ivote.html`: หน้าคูหาลงคะแนนเสียงอิเล็กทรอนิกส์หลังตรวจสอบสิทธิ์ผ่าน
    *   `results.html`: หน้าแสดงผลลัพธ์คะแนนเลือกตั้งแบบเรียลไทม์หลังปิดหีบ
    *   `backend.html`: ระบบควบคุมการลงคะแนนและตรวจสอบข้อมูลเบื้องหลังสำหรับผู้ดูแล

## 🛠️ เทคโนโลยีที่ใช้พัฒนา (Technology Stack)

### โครงสร้างพื้นฐานหลัก (Root Project)
*   **Frontend:** HTML5 (โครงสร้างหน้าเว็บแบบ Semantic HTML), CSS3 (สไตล์ชีทแบบจัดรูปแบบตามโมดูลพร้อม Responsive Web Design), JavaScript (ES6+)
*   **Styling & Frameworks:** Vanilla CSS (ไม่มีการใช้ Tailwind ในหน้าหลักเพื่อประสิทธิภาพสูงสุดในการโหลด), Bootstrap 5 (ในบางส่วน เช่น Modal แจ้งเตือน)
*   **Authentication:** Auth0 SPA SDK สำหรับเชื่อมต่อระบบยืนยันตัวตนด้วย Google Workspace ของมหาวิทยาลัย
*   **PWA Support:** Progressive Web App ในตัว มีไฟล์ `manifest.json` และ `service-worker.js` ช่วยให้ผู้ใช้สามารถติดตั้งแอปพลิเคชันลงบนหน้าจอหลักของโทรศัพท์มือถือหรือคอมพิวเตอร์ได้เหมือน App ปกติ
*   **Backend Integration:** ไฟล์ Google Apps Script (`api/auth0_m2m.gs`) สำหรับประมวลผลหลังบ้านร่วมกับ Google Sheets หรือบริการภายนอกอื่นๆ

### แอปพลิเคชัน React (`blocks-app/`)
*   โฟลเดอร์สำหรับระบบ/โปรโตไทป์ย่อยที่ขับเคลื่อนด้วย React
*   **Framework:** Vite + TypeScript
*   **Styling:** Tailwind CSS + `tailwindcss-animate`
*   **UI Components:** Radix UI (`@radix-ui/react-dialog`, `@radix-ui/react-slot`), Lucide React Icons

## 💻 การติดตั้งและเปิดใช้งานโปรเจกต์ (Installation & Running)

### 1. การเรียกใช้งานหน้าเว็บหลัก (Root static site)
โปรเจกต์หลักเป็นแบบ Static Web Page (HTML/CSS/JS) คุณสามารถเปิดใช้งานบนคอมพิวเตอร์ของคุณได้โดยไม่ต้องติดตั้ง Node.js หรือผ่าน Build Tools ใด ๆ
*   ใช้ Extension เช่น **Live Server** บน VS Code
*   หรือเปิดผ่านหน้าเว็บ GitHub Pages ที่เชื่อมไว้: `https://solution.litalkeducation.com/engedlru/`

### 2. การเปิดใช้งานระบบย่อย React (`blocks-app`)
หากต้องการพัฒนาหรือรันระบบในโฟลเดอร์ `blocks-app` ให้ใช้คำสั่งผ่าน Terminal ดังนี้:

```bash
# 1. เข้าไปยังโฟลเดอร์ blocks-app
cd blocks-app

# 2. ติดตั้ง Dependencies ที่จำเป็น
npm install

# 3. รันระบบสำหรับการพัฒนาแบบ Local (Development Server)
npm run dev

# 4. หากต้องการ Build สำหรับ Production
npm run build
```
